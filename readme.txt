# Broker Recommendations Tracker

This Python script fetches and displays the latest broker recommendations (price targets, analyst ratings) using the [Financial Modeling Prep API](https://financialmodelingprep.com/).

---

## Features

- Fetches the latest broker recommendations from Financial Modeling Prep's RSS feed.
- Automatically filters data to show only the most recent articles.
- Displays the following information for each recommendation:
  - Stock symbol
  - Publication date
  - Article title
  - Price target
  - Analyst company
  - Article URL

---

## Requirements

- Python 3.x
- A valid API key for Financial Modeling Prep
